## Admin Feature Families

- Platform governance and configuration control
- Account and team administration
- Challenge authoring, deploy, and version management
- Runtime monitoring and intervention (`Monitoring`, pod logs, request logs)
- Submission moderation and scoreboard administration
- Reward engine workflows and analytics export
- Ticket response and compliance-grade audit surfaces

## Contestant Feature Families

- Login and authenticated team context
- Challenge discovery by category/topic
- Requirement-aware challenge access and lock states
- Deploy/runtime start-stop lifecycle
- Token-based HTTP/TCP challenge access
- Flag submission with attempts and cooldown constraints
- Hint unlock flow with score cost and prerequisites
- Instance management (`GO`, `STOP`, `REFRESH`)
- Team profile, scoreboard, and action log visibility
- Ticket create/list/view/delete (policy-constrained)
