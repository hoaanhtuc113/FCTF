# Team Collaboration and Support

Contestants coordinate through profile context, standings visibility, team action history, and the built-in support ticket system.

## Team Profile and Progress Context

The profile page provides participant and team context in one view.

![Contestant profile and team stats](/img/contestant/ContestantProfile.png)

Contestants can:

- Review username, team, and member list
- Track team rank and points
- Monitor team score progress versus total challenge points
- Change password using the secure profile action

## Scoreboard and Competition Awareness

Contestants can evaluate competition status from private and public scoreboard surfaces.

![Contestant scoreboard view](/img/contestant/ViewScoreboardPage.png)

Key uses:

- Track rank shifts and score evolution charts
- Compare team position against nearby competitors
- Align challenge priorities based on standings gap

## Team Action Logs

The `Action Logs` page helps teams coordinate by exposing time-ordered actions.

![Team action logs page](/img/contestant/ActionsLogPage.png)

Contestants can:

- Filter by action type and topic
- Search by user/topic/detail text
- Review recent actions such as challenge start and flag outcomes

## Ticket Support Workflow

Contestants can submit support requests directly from the Tickets module.

![Contestant ticket list](/img/contestant/ContestantTicketPage.png)

Create flow:

![Create ticket dialog](/img/contestant/CreateTicket_popup.png)

Supported operations:

- Create a ticket with `Title`, `Type`, and `Description`
- View owned ticket details and organizer replies
- Delete eligible tickets (open tickets without organizer reply)
- Track ticket status transitions (`Open`/`Closed`)
